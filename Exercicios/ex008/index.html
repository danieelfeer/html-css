Tipos de Formatação

Negrito / Destaque

Nesta frase temos o termo destacado em negrito com a tag "b" (sem semântica).Não Recomendado

Nesta frase temos o termo destacado em negrito com a tag "strong"(com semântica). Recomendado

Itálico / Ênfase

Nesta frase temos o termo destacado em itálico usando a tag "i" (sem semântica). Não Recomendado

Nesta frase temos o termo destacado em itálico usando a tag "em"(com semântica). Recomendado

Marcador de texto

Neste parágrafo temos a maração do texto usando a tag "mark"!

Nesse texto marcado, mudamos a cor aplicando CSS no head!

Texto GRANDE & pequeno

Podemos deixar o texto grande aplicando a tag "big"(não semântica).

Para deixar o texto com letras menores, utilizamos a tag "small"

Texto deletado

Para deixar o texto como se fosse deletado, como se fosse rasura, aplicamos a tag "del".

Texto inserido

Para mostrar que inserimos um texto utilizamos a tag "ins"

Também tem a tag "u"(não semântica), que sublinha o texto.

Texto sobrescrito

A sobreposição de um texto é bastante comum para equações de matemática e utilizamos a tag "sup" por exemplo: X20+4

Texto subescrito

A subreposição de um texto é utilizado da seguinte maneira: H2O.
