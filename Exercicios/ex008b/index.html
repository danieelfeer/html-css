Outras Formatações

Para deixar um link de um documento mais visivel (utilizando fonte monoespaçada) para leitrura usamos a tag "code" como aqui: doc.type(Pyton).c

Para colocar um código inteiro, colocamos a tag "pre" junto com a "code". Como por exemplo:

        
int num;
scanf("%d",num);

if(num%2==0){
    printf("Eh Par!\n");
} else{
    printf("Eh Impar!")
}
